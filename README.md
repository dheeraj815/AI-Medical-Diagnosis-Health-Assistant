# 🏥 AI Medical Diagnosis Health Assistant

An AI-powered healthcare assistant designed to support preliminary medical
diagnosis and health monitoring using machine learning and intelligent
data analysis.


## 🚀 Live Demo

🔴 https://ai-medical-diagnosis-health-assistant.streamlit.app


## ✨ Features

- 🩺 Symptom based disease prediction
- 📊 Confidence score for each prediction
- 💊 Precaution & medicine recommendations
- 📈 Visual charts for top predictions
- ⚡ Real-time results


## 🧠 Tech Stack

- Python
- scikit-learn
- Pandas
- NumPy
- Streamlit


## 📊 Models Used

- Random Forest Classifier
- Decision Tree
- Logistic Regression
- Naive Bayes


## ▶️ Run Locally

git clone https://github.com/dheeraj815/AI-MEDICAL-DIAGNOSIS-HEALTH-ASSISTANT.git
cd AI-MEDICAL-DIAGNOSIS-HEALTH-ASSISTANT
pip install -r requirements.txt
streamlit run app.py
